# ✋ Hand Digit Recognition System

## 📘 Project Overview
This project is a real-time Hand Digit Recognition System that detects and counts fingers using Computer Vision and MediaPipe. It supports dual-hand tracking and displays the total number of detected fingers live through a webcam feed.

## 🚀 Features
* Real-time hand and finger detection
* Supports detection of both hands simultaneously
* Live finger counting using webcam
* Accurate landmark tracking with MediaPipe
* Interactive OpenCV display window
* Smooth and user-friendly visualization

## 🛠️ Tools & Technologies Used
* Python
* OpenCV
* MediaPipe
* Computer Vision
* Real-Time Image Processing

## 📂 File Included
* HAND_DIGIT_RECOGNITION.py

## ⚙️ How It Works
* Captures live video from webcam
* Detects hand landmarks using MediaPipe
* Identifies raised fingers based on landmark positions
* Counts total fingers from both hands
* Displays live output on screen in real-time
